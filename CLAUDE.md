# Claude Code 指示ファイル — 3D-bunkasai-2026

## 必須：作業開始前に必ず読むこと

作業を始める前に、必ず以下を **この順番で** 読むこと：

```
1. プロジェクト概要.md   ← 基本情報・フォルダ構成・重要ルール・技術仕様・次の課題
2. 生徒用/変更点.md      ← 直近の変更履歴（上の3〜5エントリを確認）
```

`プロジェクト概要.md` の「現在の状態・引継ぎ情報」セクションに技術仕様や演出上の決定事項が記載されている。  
`変更点.md` で直近に何が変わったかを把握してから作業すること。読まずに作業を始めてはならない。

## 作業中に守ること

- ファイルを上書き・削除する前に `過去版/` へコピーしてアーカイブする（命名：`元ファイル名_YYYYMMDD.拡張子`）
- 作業後は `生徒用/変更点.md` に時刻付きで記録する
- ユーザーから新しいルール・方針・決定事項を伝えられたら `プロジェクト概要.md` の重要ルールに追記する
- 上記の変更はすべてコミット・GitHubへプッシュする
