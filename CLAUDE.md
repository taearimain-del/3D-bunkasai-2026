# Claude Code 指示ファイル — 3D-bunkasai-2026

## 必須：作業開始前に必ず読むこと

作業を始める前に、必ず最初に以下のファイルを読むこと：

```
プロジェクト概要.md
```

このファイルにはプロジェクトの基本情報・フォルダ構成・重要ルール・次の課題が記載されている。読まずに作業を始めてはならない。

## 作業中に守ること

- ファイルを上書き・削除する前に `過去版/` へコピーしてアーカイブする（命名：`元ファイル名_YYYYMMDD.拡張子`）
- 作業後は `生徒用/変更点.md` に時刻付きで記録する
- ユーザーから新しいルール・方針・決定事項を伝えられたら `プロジェクト概要.md` の重要ルールに追記する
- 上記の変更はすべてコミット・GitHubへプッシュする
