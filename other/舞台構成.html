2. 上手（かみて）・下手（しもて）の定義

本公演での統一定義

客席から舞台（黒板方向）を向いたとき、右手が上手、左手が下手。
台本・役割書に書かれている「上手」「下手」はすべてこの基準に従っている。
  • 右手 → 上手（かみて）
  • 左手 → 下手（しもて）
⚠️ 実際の教室で確認すること：廊下側と窓側のどちらが上手（右）になるかは、教室の向きによって変わる。
確認後、下記のパターンからどちらかを採用して ⚠️ を削除する。
パターン上手（右）下手（左）
A案（廊下側が上手） 廊下・入口側 窓側
B案（窓側が上手） 窓側 廊下・入口側

3. 照明パネル（蛍光灯スイッチ）の位置

⚠️ 要現地確認。一般的には入口ドア付近の壁に設置されていることが多い。
  • 照明係の立ち位置：パネルから直接操作できる位置。客席後方の壁際 or ドア付近。
  • 照明係2〜3名が手元に台本（キューシート）を持って待機する。
  • 舞台監督の手信号が見えるように、舞台全体への視野を確保した位置に立つこと。

4. 各ACTの大道具・役者配置

ACT1　幼い日の姉妹

項目内容
**背景パネル** なし（暗転明けで部屋のイメージ）
**大道具** 上手に椅子または低いベンチ（エルサのベッド）、下手に同様（アナのベッド）
**主な役者の位置** 子供エルサ：上手ベッド、子供アナ：下手ベッド → 中央へ移動
**備考** 茶系背景パネルは暗転後のACT2に合わせてセット

ACT2　戴冠式とハンスとの出会い

項目内容
**背景パネル** 茶系背景パネル（城内の壁・城の厳かな雰囲気）
**大道具** なし（背景パネルのみ）
**主な役者の位置** 戴冠式：大人エルサ中央奥、司祭中央手前、夏アナ上手寄り後方、家来数名は舞台奥または袖。舞踏会：大人エルサ中央、夏アナとハンスは下手寄り一角
**備考** 杖・玉を大人エルサ登場位置にセット済み。白手袋は開演30分前に渡し済み

ACT3　エルサの逃走

項目内容
**背景パネル** 茶系パネルのまま（舞踏会継続） → 「レット・イット・ゴー」歌い出しに合わせて白・水色パネルを後方に展開
**大道具** 「レット・イット・ゴー」開始時：白・水色の布/パネルを上手の大道具2名が展開
**主な役者の位置** エルサ中央やや奥、アナ・ハンスは下手寄り、公爵は上手端。逃走後：エルサのみ上手袖から再登場し舞台奥中央に立つ
**備考** ドアフレームはこのACTでは使わない。暗転後に白・水色パネル撤去＋ドアフレームを上手袖の奥から搬入

ACT4　クリストフ・オラフとの旅

項目内容
**背景パネル** なし（パネル撤去済み・野外のイメージ）
**大道具** ドアフレーム：舞台中央やや奥に設置（納屋の扉として使用）
**主な役者の位置** クリストフ：ドアフレーム内側（奥側）、スヴェン：フレーム横、アナ：小走りで登場後ドアをノック。オラフは袖または小道具の陰からひょっこり登場
**備考** ロープ・金づち・人参をアナの手が届く位置に配置済み。オラフの人参（鼻）は暗転中に装着

ACT5　エルサの城・トロールの村

項目内容
**背景パネル** 白・水色パネル（氷の城・ACT3と同じパネルを再利用）
**大道具** 白・水色パネルを前面に出す。ドアフレームは撤去済み
**主な役者の位置** エルサ：上手袖から先行入場→舞台奥中央。冬アナ・クリストフ・オラフ・スヴェン：下手から登場。トロール村：トロール2〜3人ずつ両袖から登場
**備考** ACT5→ACT6の45秒暗転が最重要。エルサの横断動線（上手袖→舞台裏→下手袖）を確保すること

ACT6　ハンスの裏切り〜フィナーレ

項目内容
**背景パネル** 茶系背景パネル（ハンスの部屋・室内）。クライマックス後も残す（城の外壁として転用）
**大道具** 教室の机1台（執務机）＋椅子1脚：舞台中央やや奥。クライマックス前の暗転で上手袖へ撤収
**主な役者の位置** ハンス：机の前の椅子に着席。冬アナ：息を切らせながら駆け込み。オラフ：袖から駆け込み。クライマックス：エルサ下手袖→中央、ハンス上手から剣持参、冬アナ・クリストフ・スヴェン・オラフ下手から
**備考** 暖炉プロップ（LED点灯済み）を舞台端壁際に設置。机の上に書類・地図・羽ペンを配置。ハンスの剣はクライマックス前暗転で手渡し

5. 舞台袖の使い方

上手袖（教室内・黒板に向かって右側）

#### 【上手袖の「奥」とは】

上手袖の中でもさらに舞台奥（黒板側）寄りのエリアを「奥」と呼ぶ。

```

┌──────────────────────────┐

黒板方向 │ │ ← 「奥」エリア（エルサが通過する中継地点）

（上） │ ← エルサの動線 → │

│ ← 大道具スタンバイ位置 → │ ← エルサ動線より舞台裏側に配置

│ （机・椅子・ドアフレーム）│

│ │

│ 上 手 袖 │

└──────────────────────────┘

客席方向 →（舞台との境界）

```

  • 大道具の机・椅子・ドアフレームは「エルサ動線より舞台裏側」にスタンバイさせること
  • 「エルサ通過確認後」に初めて搬入を開始する（動線がかぶると衝突するため）

#### 主な用途

  • 大人エルサの出入り（ACT5→ACT6の舞台横断の出発点）
  • ドアフレームの事前スタンバイ（ACT4用・奥に配置）
  • 机・椅子の事前スタンバイ（ACT6用・エルサ動線より奥に配置）
  • 大道具係2名の待機・「レット・イット・ゴー」パネル展開担当
  • ACT3：ウェーゼルトン公爵・家来の退場方向
  • ACT7クライマックス：ハンスが剣を受け取りスタンバイ

下手袖（教室内・黒板に向かって左側）

  • 主な用途
  • 冬アナ一行（クリストフ・スヴェン・オラフ）のスタンバイ（ACT5〜6）
  • 小道具係の担当エリア（暖炉プロップのセット等）
  • エルサのACT5→ACT6横断の到着点・待機（下手袖から舞台中央へ移動）
  • ACT7フィナーレ：冬アナ・クリストフ・スヴェン・オラフが下手から登場
  • ACT5→ACT6暗転：小道具係が下手袖から書類・地図・羽ペンをセット（エルサは上手寄り1.5m、小道具係は下手寄りで並行作業）

廊下（使える場合のみ）

⚠️ 廊下が使えるか・横幅は要確認。使える場合は上手袖の延長として活用可能。

6. ACT5→6 暗転（45秒）エルサ動線

この暗転は全公演中で最も複雑。エルサの動線を最優先に確保すること。

```

ACT5→6 暗転（45秒）のエルサ動線：

[上手袖（奥）] ─────→ [舞台裏を横断] ─────→ [下手袖（待機）] ─→ [舞台中央]

0〜 5秒：進入 5〜15秒 〜40秒 35〜40秒

↓ エルサが上手袖を通過したのを確認してから ↓

大道具係が机・椅子の搬入を開始（20〜35秒）

↓ 同時進行 ↓

大道具係4名：氷パネル撤去→茶系パネルに切替（0〜25秒）

小道具係2名：下手袖から書類・地図・羽ペンをセット＋暖炉プロップ設置（20〜35秒）

```

重要ルール：

1. 上手袖担当が「エルサ通過確認」を舞台監督に肉声で報告する

2. 報告を受けてから舞台監督が「机・椅子搬出GO」の合図を出す

3. 35秒時点でパネル切替未完了なら即「延長合図（手を2回叩く）」→最大+15秒（計60秒）

4. ゲネプロで必ず計測・記録すること
